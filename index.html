Recent Project

  • Your Personal Voice Assistant: Using Python

INTRODUCTION


This is a Python script that utilizes various libraries to create a voice-controlled assistant named "Peter." The script uses `pyttsx3` for text-to-speech conversion, `speech_recognition` for voice recognition, and other libraries for additional functionalities. "Peter" can answer questions about its name, age, and creator, tell the current time, open YouTube and Google, share jokes, play music, and exit when commanded. The script listens for voice commands and responds accordingly. It's a simple voice assistant that demonstrates voice interaction with a computer program, providing convenience and entertainment.


TECHNOLOGIES USED


  • pyttsx3 - This library is used for text-to-speech (TTS) conversion. It allows the program to convert text into spoken words.
  • webbrowser - This library is used to open web browsers and navigate to specific websites, such as YouTube and Google.
  • pyjokes - This library provides jokes in various categories and languages. It is used to fetch and tell jokes.
  • speech_recognition - This library provides speech recognition capabilities. It allows the program to listen to spoken words and convert them into text.
  • datetime - The datetime module is used to retrieve the current date and time, which can be spoken out loud.
  • os - The os module is used for interacting with the operating system. In this program, it is used to play music files and list files in a directory.
